# Changelog

## [1.0.0] - 2026-05-29

### Added
- Sidebar chat panel with streaming responses
- Multi-model support: DeepSeek-R1, DeepSeek-Coder, DeepSeek-V4-Pro
- Code actions: Explain, Refactor, Fix Bugs, Generate Comments, Generate Unit Tests
- Right-click context menu integration
- Command palette commands
- Git commit message generation (SCM integration)
- Custom API endpoint configuration (supports private deployments)
- Configurable API Key, temperature, max tokens
- Keyboard shortcuts
- Webview chat with markdown rendering
